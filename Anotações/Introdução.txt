Estudos sobre a biblioteca React


REACT - uma biblioteca javascript para criar interfaces de usuários

{
    Criado em 2001 por Jordan Walke no Facebook, baseado no XHP, um framework para criação de HTML
    no php, utilizado no mudal de notícias de ferramenta


    2012 - Utilizado no Instagram
    2013 - Anúncio para liberação OPenSource na JSConfUs
    2015 - React Native
    2015 - UWP (Universal Windows Platform)


    A principla função é a criação da interface para o usuário

    React é uma linguagem declarativa

    Pode ser utilizado em qualquer lugar 
}
